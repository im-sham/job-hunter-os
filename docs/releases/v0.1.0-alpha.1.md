# Job Hunter OS v0.1.0-alpha.1

This is the first public alpha release of `Job Hunter OS`.

## Recommended Install Path

For most testers, use the macOS `.dmg`.

- Primary installer: macOS DMG
- Fallback installer: macOS zip

## What This Alpha Includes

- local-first workspace setup
- guided onboarding from resumes, LinkedIn exports, bios, and writing samples
- job targets and reusable application details
- assistant-guided sourcing into a review queue
- pipeline approval and assistant task packaging
- application prep with human-gated answers
- local browser assist for common ATS flows

## Best Fit For This Release

This alpha is best for:

- design partners
- early testers
- users willing to give structured feedback on install, UX, sourcing, and application workflows

## Important Notes

- The desktop app is currently macOS-only.
- The build is unsigned and not notarized, so macOS may ask you to right-click and choose `Open` on first launch.
- This is still an alpha product. Some flows are already strong, but broader ATS coverage, packaging polish, and assistant smoothness are still being improved.
